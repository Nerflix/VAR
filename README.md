# VAR
Python econometrics project implementing a Vector Autoregression (VAR) model to forecast financial asset returns. Uses multiple macroeconomic indicators and ROI metrics as endogenous variables. Includes stationarity testing (ADF), lag order selection (AIC/BIC), impulse response analysis, and forecast visualisation with pandas and statsmodels.
